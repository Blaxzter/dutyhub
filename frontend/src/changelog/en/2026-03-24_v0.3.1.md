## Better Mobile Layout

The entire app layout has been reworked for smaller screens. Navigation menus, event views, booking pages, and settings all adapt more cleanly to phones and tablets now. If you've been using the app primarily on a mobile device, this update should feel like a significant step up in usability — less horizontal scrolling, better touch targets, and a layout that actually fits your screen.
