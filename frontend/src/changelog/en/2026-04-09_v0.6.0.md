## Group Managers

Admins can now assign dedicated managers to individual event groups. A group manager can create and edit events, view member availability, and access reports — all scoped to the groups they are responsible for, without needing full admin rights.

Open any event group, switch to the new Management tab, and search for a user to add them as a manager.

![Group Managers tab showing assigned managers with add and remove options](./images/group-managers.png)

## Redesigned Event Group Detail

The event group detail page has been completely reworked. On desktop, a tabbed side navigation lets you switch between Events, Availability, and Management sections. On mobile, the same sections are accessible via swipeable chips at the top of the page.

![Redesigned event group detail page with tabbed side navigation](./images/event-group-detail.png)

## Reporting for Group Managers

The Reports page is no longer limited to admins. Group managers can now access it too, with data automatically filtered to the events and groups they manage.

## Sidebar Status Badges

Event groups and events that are not yet published now show a small status badge in the sidebar. This makes it easy to tell at a glance which items are still in draft or archived — and a tooltip reminds you that they are only visible to managers.

![Sidebar showing draft badges on unpublished event groups and events](./images/sidebar-status-badges.png)
