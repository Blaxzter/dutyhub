## Dienstplan drucken

Sie brauchen eine Papierversion Ihrer anstehenden Dienste? Es gibt jetzt eine eigene Druckansicht, die Ihren Dienstplan in einem übersichtlichen, druckfreundlichen Layout formatiert. Öffnen Sie sie über die Dienstplanseite, drücken Sie auf Drucken — fertig. Perfekt zum Aufhängen am Schwarzen Brett oder für Ihren Planer.

## Mehrsprachige Benachrichtigungen

Bisher wurden Benachrichtigungen und Systemmeldungen immer auf Englisch verschickt, unabhängig von Ihrer Spracheinstellung. Ab dieser Version berücksichtigen alle Benachrichtigungen Ihre gewählte Sprache. Wenn Sie die App auf Deutsch eingestellt haben, erhalten Sie Buchungsbestätigungen, Erinnerungen und andere Nachrichten ab sofort ebenfalls auf Deutsch.
